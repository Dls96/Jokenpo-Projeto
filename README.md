* Android Studio 3.5 com:
    * SDK Platform com SDK 4.0 ou mais
    * Ultima Versão do Android SDK Build-Tools
    * Android Emulator 
    * Android SDK Platform-Tools
    * Android SDK Tools
    * Intel x86 Emulator Accelerator (HAXM installer)
